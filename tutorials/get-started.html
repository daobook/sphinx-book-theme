开始行动¶

这是一个开始使用 Sphinx 图书主题的循序渐进的简短教程。

备注

此文档和下面的例子是用 MyST Markdown 写的，这是一种可以和 Sphinx 一起使用的 Markdown 形式。关于 MyST markdown 以及在 Sphinx 网站如何使用 MyST Markdown 的更多信息，请看 MyST-parser 文档。

先决条件¶

你应该比较熟悉 Sphinx 生态系统，并在你的电脑上本地安装了它。

备注

如果你还没有一个可供定制的 Sphinx 网站，可以用以下方法创建一个：

sphinx-quickstart

安装并激活主题¶

首先用 pip 安装 sphinx-book-theme。

pip install sphinx-book-theme

然后，在你的 Sphinx 配置（conf.py）中激活该主题。

...
html_theme = "sphinx_book_theme"
...

这将为你的文档激活 Sphinx 图书主题。

备注

你可能需要注释关于 html_theme_options 的配置，这取决于你以前的主题。

在你的主题中添加一个源码库按钮¶

有几种方法可以定制 Sphinx 图书主题。在本教程中，将为我们的主题添加一个指向 GitHub 仓库的指针。

要添加一个链接到文档库的按钮，请在 conf.py 中添加以下配置。

html_theme_options = {
    ...
    "repository_url": "https://github.com/{your-docs-url}",
    "use_repository_button": True,
    ...
}

当你构建你的文档时，你的顶栏现在应该包括一个小的 GitHub 标志，它有一个到仓库的链接。

参见

更多信息见 添加一个链接到你的存储库。

定制你的左侧边栏¶

在页面上有几个主要的区域，你可以用图书主题进行定制。最常见的是 主侧边栏（默认情况下，在页面的左边）。

侧边栏元素被 Sphinx、Sphinx 插件以及当前主题定义为 HTML 模板。你可以使用 html_sidebars 配置指定哪些页面应该包含哪些侧边栏元素。本教程中我们将跳过这一步，但如果你想了解更多，请看 控制左边的侧栏项目。

让我们 定制网站的标志。为此，我们将编辑 html_logo 和 html_title 配置选项。

要添加网站标识，请遵循以下步骤：

  1. 在你的文档的根部放一张图片（例如，myimage.png）。

  2. 在你的 conf.py 文件中添加一行，看起来像这样。

    html_logo = "path/to/myimage.png"

    其中路径是相对于你的 conf.py 文件而言的。

接下来我们将 自定义网站的标题。要做到这一点，在你的 conf.py 文件中添加一行，看起来像这样：

html_title = "My site title"

这个标题现在将被放在标志的下面。

在页面上添加一些外边距内容¶

这个主题支持几种特殊的内容，主要是受 Tufte CSS 主题样式 的启发。一种内容是 外边距内容。这允许你把内容 “弹出” 到页面的页边距。通过添加以下指令（用 MyST Markdown 编写）在页面上添加一些外边距内容。

Look, some margin content!

On wider screens, this content will pop out to the side!

```{margin} Look, some margin content!
On wider screens, this content will pop out to the side!
```

在宽屏上，外边距内容将弹出到页面的一侧，并允许它下面的内容向上移动。这允许你在不中断信息流的情况下提供额外的信息”

你可以用 Sphinx 图书主题做很多其他事情。现在你已经有了一个开始，请查看左边的其他部分，了解关于如何使用它的更多信息。
