sphinxcontrib-bibtex - References and bibliographies¶

sphinxcontrib-bibtex provides support for citations and bibliographies.

Because this theme deals with scholarly and scientific communication, sphinxcontrib-bibtex should work particularly well.

Here’s are a few citations:

  • Default citation [Project Jupyter et al., 2018], [Holdgraf et al., 2016, Project Jupyter et al., 2018]. (should be same as parenthetical)

  • Parenthetical citation [Project Jupyter et al., 2018], [Holdgraf et al., 2016, Project Jupyter et al., 2018].

  • Textual citation Project Jupyter et al. [2018], Holdgraf et al. [2016], Project Jupyter et al. [2018].
