侧边栏¶

sphinx-book-theme 中有两种不同的类似侧边栏的内容，典型的 {sidebar} 指令，以及一个特定主题的 {margin} 指令。本节涵盖这两种指令。两者都允许你将额外的内容与你的主要内容分开放置。

小技巧

侧边栏内容一般会与你的网站目录所在的白色空间重叠。当读者滚动侧边栏内容进入视野时，右边的 TOC 应该自动隐藏起来。

页边距内容¶

你可以指定应该存在于右页边距的内容。这将表现得像一个普通的侧边栏，直到屏幕达到一定的宽度，在这一点上，这些内容将 “弹出” 到右边的页边距区域。

有两种方法可以将内容添加到空白处：通过 {margin} 指令，以及通过向自己的内容添加 CSS 类。

使用 {margin} 指令来添加页边距内容¶

{margin} 指令允许你用自己的标题和内容块来创建边缘内容。它是Sphinx {sidebar} 指令的一个封装器，主要通过 CSS 类来实现其魔力（见下文）。

下面是如何使用 {margin} 指令的：

Here is my margin content

It is pretty cool!

```{margin} **My margin title**
Here is my margin content, it is pretty cool!
```

使用 CSS 类来增加页边距内容¶

你也可以直接给你的页面上的元素添加 CSS 类，以使它们表现得像页边距内容。要做到这一点，请将 margin CSS 类添加到页面上的任何元素。许多 Sphinx 指令允许你直接添加类。例如，下面是在 {note} 指令中添加 margin 类的语法：

注解

这个注解将写在页边距！

:::{note}
:class: margin
This note will be in the margin!
:::

对页面上的大多数元素都有效，但一般来说，对作为内容束最顶端的元素的 “父容器” 效果最好。

例如，我们甚至可以像这样把一个完整的数字放在页边距：

如果你愿意，你也可以把整个数字写在页边距。这里是一个数字，下面有一个标题。我们在下面加一个说明，创造一些垂直空间，以便看得更清楚。

```{figure} images/cool.jpg
---
figclass: margin
alt: My figure text
name: myfig4
---
And here is my figure caption
```

我们可以用 这是我的图片说明 来引用该图。或者用 图 1 这样的编号引用。

内容侧栏¶

内容侧栏与你的文本并列存在，但允许页面的其他部分围绕它们流动，而不是移动到右边的空白处。要添加内容侧边栏，请使用以下语法：

我的侧边栏标题

注解

这是我的侧边栏内容，它是相当酷的！

注意内容是如何围绕着侧边栏向右包裹的。然而，侧边栏的文字仍将与你的内容保持一致。有一些元素，如 “note” 块和代码单元，可能会与侧边栏发生冲突。如果发生这种情况，可以尝试使用 {margin} 来代替。

```{sidebar} **My sidebar title**
Here is my sidebar content, it is pretty cool!
```

全宽内容¶

全宽内容延伸到右边的空白处，使它在你的书的其他内容中脱颖而出。要在你的页面上添加全宽内容，请在文档中的任何元素上添加 full-width 类。例如，你可以像这样给笔记元素添加一个 full-width 标签：

```{note}
:class: full-width
This content will be full-width
```
