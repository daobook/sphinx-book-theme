加入你自己的 CSS 规则¶

要进一步定制你的网站的外观，你可以如 ReadTheDocs Docs 中所述，定制你的 CSS 样式表。

首先，创建一个 CSS 文件，并将其放在 _static/custom.css 中。一个改变顶层标题颜色的 CSS 文件的例子可能看起来像这样。

h1 {
    color: #003B71 !important;
}

你还需要在你的 conf.py 文件中加入这两行内容

html_static_path = ["_static"]
html_css_files = ["custom.css"]
