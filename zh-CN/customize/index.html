定制¶

这些部分描述了一些你可以定制你的主题的外观和感觉的方法。

主题选项¶

以下选项可通过 html_theme_options 使用

键

类型

描述

single_page

布尔值

去掉左边的侧边栏，把网站当作一个单页。见 使用该主题的单页版本。

path_to_docs

字符串

文档的路径，相对于版本库根目录（例如：docs/）。参见 添加按钮以链接到你的来源。

repository_url

字符串

文档存储库的 URL（例如 GitHub 存储库的URL）。参见 添加一个链接到你的存储库。

repository_branch

字符串

文档库的分支（例如：master', main’, `docs’）。见 添加一个链接到你的存储库。

use_issues_button

布尔值

在导航条添加一个按钮，链接到版本库的议题（与 repository_url 和 repository_branch 一起使用）。见添加一个链接到你的存储库。

use_download_button

布尔值

在导航栏添加一个按钮，以下载页面的源文件。见 添加按钮以链接到你的来源。

use_fullscreen_button

布尔值

在导航栏添加一个按钮来触发全屏模式。

use_repository_button

布尔值

在导航栏中添加一个链接到文档库的按钮。见 添加一个链接到你的存储库。

launch_buttons

布尔值

包含从 Jupyter 笔记本建立页面的 Binder 启动按钮。参见 用于交互的启动按钮。

home_page_in_toc

布尔值

是否将主页放在导航栏（顶部）。见 将主页添加到你的目录中。

logo_only

布尔值

只显示标识，不显示 html_title，如果它存在的话。

show_navbar_depth

整数

在导航栏中显示子目录，向下到这里列出的深度。见控制左侧侧边栏列表的展开深度.

extra_navbar

字符串

在侧边栏页脚添加额外的 HTML（如果 sbt-sidebar-footer.html 在 html_sidebars 中被使用）。见 自定义侧边栏页脚。

extra_footer

字符串

在每个页面的页脚添加额外的 HTML。

toc_title

字符串

将与页内 TOC 一起显示的文本（默认为 Contents）。

theme_dev_mode

布尔值

（仅限开发人员）触发一些功能，使开发主题更容易。
