定制二级侧边栏¶

二级侧边栏包含关于当前页面的信息。它从页面的顶部开始（在顶栏），并向下延伸（默认情况下，从页面的右侧开始）。本页描述了控制和定制二级侧边栏的方法。

重命名页内目录¶

你可以重新命名页面内目录的标题，在右侧边栏：

html_theme_options = {
    "toc_title": "{your-title}"
}

标题的默认值是 Contents。

默认显示更多层次的页内 TOC¶

通常情况下，在页内目录中只显示一个页面的第 2 级标题，更深的级别只有在它们是活动部分（在屏幕上滚动时）才会显示。

你可以通过使用以下配置，默认显示更深的层次，表明应该显示多少个层次：

html_theme_options = {
  "show_toc_level": 2
}

现在，无论页面上显示的是什么，指定级别以下的所有标题都将被显示。
