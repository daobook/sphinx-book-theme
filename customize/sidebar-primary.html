定制主侧栏¶

主侧边栏一般包含网站导航和标志。默认情况下，它位于网站的左侧。本页描述了你可以控制和定制主侧边栏的方法。

控制左边的侧栏项目¶

你可以控制什么样的内容放在左上方你的网站的标志和名称下面。

要做到这一点，在你的 conf.py 文件中使用 html_sidebars 变量。这个变量以文件名模式的字典为键，以侧边栏元素的列表为值。任何与键相匹配的文件都会有相应的侧边栏元素放在该页面的侧边栏中。

例如，以下配置将在 posts/ 文件夹下的页面上 只包括页脚：

html_sidebars = {
    "posts/*": ["sbt-sidebar-footer.html"]
}

你也可以使用 ** 将一组侧边栏应用于你的书的所有页面。例如：

html_sidebars = {
    "**": ["sbt-sidebar-nav.html", "sbt-sidebar-footer.html"]
}

更多信息请见 Sphinx HTML 侧边栏文档。

默认的侧边栏元素¶

默认情况下，该主题在所有页面上都启用了这三个特定主题的侧边栏元素：

  • sidebar-logo.html：显示标志和网站标题。

  • search-field.html: 一个基于 bootstrap 的搜索栏（来自 PyData Sphinx Theme）

  • sbt-sidebar-nav.html: 一个用于你的书基于 bootstrap 的导航菜单。

  • sbt-sidebar-footer: 一个 可配置的 HTML 片段，用于添加到侧边栏（默认情况下，它被放置在底部）。

自定义侧边栏页脚¶

你可以选择你自己的 HTML，包括在你的侧边栏的页脚（或将其设置为空）。要做到这一点，请在 conf.py 中设置以下选项：

html_theme_options = {
    ...
    "extra_navbar": "<p>Your HTML</p>",
    ...
}

这段文字将被默认放置在侧边栏的底部。

为你的目录添加一个标题¶

“如果你想在 TOC 链接的某一节上面添加一个标题，请在该节的 toctree 指令中使用 :caption: 我的标题文本 为该部分添加标题。

自定义标志、标题和 favicon¶

你可以用 conf.py 中的以下 Sphinx 配置来定制你网站的标志、标题和 favicon：

html_title = "Your title"
html_logo = "path/to/logo.png"
html_favicon = "path/to/favicon.ico"

这些将被放置在你的页面的左上方。

要想 去掉标志下面的网站标题，请在 conf.py 中添加这一行：

html_theme_options = {
  ...
  "logo_only": True,
  ...
}

将主页添加到你的目录中¶

默认情况下，你的目录将从你添加到 toctree 的第一个文件开始。你也可以配置主题，在你的导航栏中也显示主题的登陆页。

要将你的网站的登陆页添加到目录中，请使用以下配置：

html_theme_options = {
    ...
    "home_page_in_toc": True
    ...
}

控制左侧侧边栏列表的展开深度¶

你可以使用 conf.py 中的以下配置，控制左侧边栏中的 toc 项目保持展开的程度：

html_theme_options = {
    ...
    "show_navbar_depth": <level>,
    ...
}

默认值是 1，它只显示文档的顶层部分（在本文档中使用）。
